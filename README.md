# Speedy Components

An AngularJs components library for the speedy project.

![Code Coverage-shield-badge-1](https://img.shields.io/badge/Code%20Coverage-100%25-brightgreen.svg)

To run tests
```
npm run test
```

To run tests with watch
```
npm run test-watch
```

To build
```
npm run build
```
